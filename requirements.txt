## 🚀 Quick Start

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/jokeratc.git
cd jokeratc

# Install dependencies
pip install -r requirements.txt

# Run JokerATC
python main.py
```

### Or install directly

```bash
pip install textual rich psutil pygame
python main.py
```

## 🎮 Controls

| Key | Action |
|-----|--------|
| `Q` | Quit JokerATC |
| `R` | Refresh all widgets |
| `T` | Cycle through themes |
| `S` | Toggle sound effects |
| `M` | Toggle matrix rain |
| `F5` | Refresh (alternative) |
| `Ctrl+C` | Force quit |

---

## 💻 Terminal Commands

| Command | Description |
|---------|-------------|
| `help` | Show all available commands |
| `clear` | Clear terminal screen |
| `sysinfo` | Display system information |
| `neofetch` | ASCII art system info |
| `theme` | List available themes |
| `theme <name>` | Switch to theme (tron/matrix/cyberpunk/amber/neon/ocean) |
| `sound` | Toggle sound on/off |
| `matrix` | Toggle matrix rain on/off |
| `reboot` | Simulate system reboot |
| `shutdown` | Exit JokerATC |
| `whoami` | Show current user |
| `uptime` | Show system uptime |
| `ps` | List running processes |
| `net` | Show network information |
| `ping <host>` | Ping a host |
| `banner` | Show JokerATC ASCII banner |

## 🐛 Troubleshooting

### Sound not working?
```bash
# Install pygame
pip install pygame

# Or disable sound in config.json
# Set "sound_enabled": false
```

### Permission errors?
```bash
# Run with sudo for full system access
sudo python main.py
```

### Display issues?
```bash
# Ensure your terminal supports Unicode
export LANG=en_US.UTF-8

# Use a modern terminal (Alacritty, Kitty, iTerm2, Windows Terminal)
```
